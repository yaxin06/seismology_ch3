3.1 Introduction

地震學如何變成地球內部的「掃描工具」？

travel time first

第三章的主軸是：利用地震波的走時、振幅與波形，反推出地球內部的速度、密度與彈性性質。地球內部不能直接挖開來看，所以地震學像一種自然版的「透視影像」。

最重要的資料是 arrival time，也就是震波到達測站的時間。若知道震源時間與位置，就能得到 travel time，再用走時去限制震波路徑上的速度構造。

Tsr = ∫ray path ds / v(s)

本章思考順序

  1. 先把地球簡化成平坦水平層。
  2. 再加入反射、折射與多測站資料。
  3. 距離變大後，改用球形地球幾何。
  4. 最後加入真實地球的非均向性與衰減。

一句話：走時提供速度構造，速度構造再連結到地球內部的溫度、礦物、成分與狀態。

3.2 Refraction Seismology

折射地震學：用第一個到的波找地下速度

Pg / Pn / PmP

折射法常把地下看成速度隨深度增加的層狀介質。最基本模型是：上層速度 v0、厚度 h0，下方半空間速度 v1，且 v1 > v0。

三種基本路徑
  • Direct wave：沿上層直接傳播，TD=x/v0。
  • Reflected wave：從界面反射回來，走時曲線是雙曲線。
  • Head wave：到達臨界角後沿高速層頂部傳播，再向上回到地表。
為什麼 head wave 可能比 direct wave 先到？

雖然 head wave 路徑比較長，但它有一段在高速層中傳播。距離夠遠時，「走高速公路繞一點路」會比「慢速直走」更快。

TH(x)=x/v1+τ1,　τ1=2h0√(1/v02 − 1/v12)

為什麼莫氏不連續面是由折射法發現的？

如果地下只有單一均勻地殼，遠距離測站不應該突然出現一條斜率較小、代表速度較快的到時線。但 Mohorovičić 觀察到遠距離 P 波比原本預期更早到，表示震波有一段沿著較深、較快的介質傳播。這個由折射定律推論出的高速界面，就是地殼與地函之間的莫氏不連續面。

折射法讀圖技巧

在走時圖上，線越平代表速度越快；截距 τ 可用來反推界面深度。若有反向測線，還可以估計傾斜界面的 dip。

實際資料常使用 reduced travel time plot，把 T−x/vred 畫出來，讓高速相位更容易比較。

3.3 Reflection Seismology

反射地震學：把地底反射面變成影像

CMP / NMO / Migration

反射法使用從地下界面反射回來的能量，常用於油氣探勘與地殼構造影像。最簡單的水平界面反射，其走時曲線是雙曲線：

T(x)2 = x2/v2 + t02

其中 t0 是零偏移距的雙程垂直走時。速度越快，雙曲線越平；反射越深，通常需要更大的 offset 才能穩定估速。

資料處理流程

  1. CMP gather：把相同中點、不同 offset 的紀錄排在一起。
  2. NMO correction：校正 offset 造成的走時差。
  3. Stacking：疊加增強真正反射，壓低雜訊。
  4. Migration：把傾斜反射與繞射能量移回正確位置。

反射剖面看起來像地底照片，但本質上是「波場在雙程走時上的顯示」。速度模型不準時，時間轉深度也會跟著偏。

3.4 Seismic Waves in a Spherical Earth

球形地球中的震波：距離一大，地球不能再當平的

p = r sin i / v

近距離折射、反射可近似平面層；但遠震穿越深部地球時，必須用球形幾何。球形地球的射線參數為：

p = r sin i / v　，且　p = dT/dΔ

速度通常隨深度增加，震波向下時會逐漸彎曲，直到入射角 i=90° 達到 turning depth，再彎回地表。

三種速度構造會造成三種走時樣貌

速度緩慢增加時，T(Δ) 連續且凹向下，p 隨距離增加而減小。

射線參數（Ray Parameter）與地震對蹠點

本節說明球形地球中射線參數 p 的物理意義，並介紹震央正對面的對蹠點概念。

p = dT/dΔ

Ray Parameter 怎麼讀？

射線參數可寫成 p = r sin i / v，也可由走時曲線斜率取得：p = dT/dΔ。同一條射線在傳播過程中 p 保持不變，因此它像是這條震波路徑的「身分證」。

p 小 → 射線通常穿得更深；p 大 → 射線較貼近淺部。

地震對蹠點 Antipode

地震對蹠點是震央在地球正對面的點，震央距為 180°。在理想均勻球形地球中，表面波會在對蹠點附近重新聚焦，所以振幅可能變大；但真實地球的橢球形狀與橫向不均質會讓聚焦變得複雜。

震央對蹠點

古氏不連續面與地震陰影帶

古氏不連續面，也就是核心—地函邊界 CMB，並不是直接被看見，而是透過遠震走時與陰影帶推論出來。

Gutenberg discontinuity

古氏不連續面位於約 2900 km 深處，是固態地函與液態外核的邊界。地震學家發現，在某些震央距範圍內，直接 P 波會明顯消失或變弱，而 S 波更無法穿越外核，這種「該出現卻沒有直接波」的區域稱為 seismic shadow zone。

G

為什麼陰影帶能指出核心邊界？

如果地球內部速度只是平滑增加，走時曲線應該較連續；但觀測顯示約 103° 到 142° 間缺少直接 P 波，代表震波遇到一個速度劇烈改變的界面並被強烈折射。再加上 S 波無法通過該深部區域，說明外核是液態，這就是推論古氏不連續面的關鍵證據。

3.5 Body Wave Travel Time Studies

體波走時研究：用相位名稱讀地球深部

P / S / K / I / J

體波相位名稱像密碼：每個字母代表震波穿過、反射或轉換的位置。透過大量地震在不同震央距的到時資料，可以建立走時曲線與地球參考模型。

符號意思
P / SP 波 / S 波穿越地函等固體區域
KP 波穿越液態外核
I / JP 波 / S 波穿越固態內核
c / i在 CMB / ICB 反射
p / s從震源向上出發後在自由表面反射的 P / S 波

相位名稱解碼

試試看：ScS、PcP、pP、SKS、Pdiff、PKiKP。

核心相位與陰影帶

液態外核的 P 波速度低於下地函，且 S 波不能穿越外核。這會造成 P 波陰影帶，並讓 PKP、PKIKP、PKiKP、Pdiff 等核心相位成為研究外核、內核與核心邊界的重要工具。

上地函 410 / 660 km 不連續面

速度快速增加會造成 triplication：同一震央距可能有三條射線、三個到時。這正是研究 410 km 與 660 km 相變界面的重要訊號。

3.6 Anisotropic Earth Structure

非均向性：同一種岩石，方向不同，速度也不同

S-wave splitting

SPO

Shape-preferred orientation：岩層、裂隙、片理等形狀排列造成整體方向性。例如水平層狀構造會使水平與垂直方向速度不同。

LPO

Lattice-preferred orientation：礦物晶格被流動或應力排列。例如橄欖石在上地函中的排列，可反映板塊運動與地函流動。

Shear-wave splitting

S 波進入非均向介質後分裂為快、慢兩個偏振方向，兩者到時差可用來推測非均向強度與快軸方向。

本節以 S 波分裂說明非均向性如何由到時差與偏振方向表現於地震紀錄中。

選擇非均向來源

非均向性通常比地球徑向速度變化小，但在研究板塊底部流動、D″ 層與內核結構時非常關鍵。

3.7 Attenuation and Anelasticity

衰減與非彈性：震波為什麼越傳越弱？

Q factor

震波振幅變小不只因為能量消失，也可能只是能量被分散到更大的波前或被聚焦到別處。講義把振幅降低分成四類：

  • Geometric spreading：波前擴大，單位面積能量變少。
  • Scattering：小尺度不均質讓能量散到其他方向，形成 coda。
  • Multipathing：速度不均造成聚焦、散焦，改變局部振幅。
  • Anelasticity：真正把彈性能轉成熱，屬於 intrinsic attenuation。

Q 值與振幅衰減

調整滑桿查看振幅比例。

震波振幅衰減的主要機制

本節區分幾何擴散、散射、多路徑與非彈性衰減，說明震波振幅隨傳播距離與介質性質改變的原因。

attenuation
幾何擴散波前變大，單位面積能量下降。
散射小尺度不均質造成 coda。
多路徑速度透鏡造成聚焦或散焦。
非彈性震波能量轉為熱，真正耗散。

高溫、部分熔融或流體通常會提高衰減，因此速度低且 Q 低的區域常被解釋為較熱或有熔體。

3.8 Composition of the Mantle and the Core

從速度、密度到成分：地函與核心是什麼做的？

PREM / density / phase change

地震學直接給我們的是 P、S 波速度與部分密度限制；要進一步說明成分，必須結合重力、地球轉動慣量、岩石物理、礦物高壓實驗與宇宙化學資料。

地球平均密度約 5.5 g/cm³，高於地表岩石密度，表示內部必須有高密度核心。外核中 S 波速度為 0，代表液態；內核有有限 S 波速度，代表固態。

重點連結

  • 410、520、660 km 不連續面：與橄欖石相變、尖晶石/鈣鈦礦結構有關。
  • CMB：密度與速度劇烈改變，是地函與外核的重大邊界。
  • 核心成分：接近鐵，但純鐵太重，因此通常推測含較輕元素。

410、520、660 km 不連續面與地函礦物相變

外核

點擊圖片即可像前面章節圖片一樣放大查看，並對照右側說明閱讀 P 波、S 波與不同能量帶。

這張圖是 2026/5/1 地震的 record section。record section 的概念是：把不同測站的地震波形依震央距排列，形成一張「距離—時間」圖。每一條 wiggle 是一個測站記錄，斜向排列的能量帶就是同一個震波相位在不同距離的到時。

  • 先找第一個清楚到時：通常是 P 波，代表最快的體波路徑。
  • 看斜率：斜率 dT/dΔ 就是射線參數 p；越平表示表面視速度越快。
  • 看相位分離：若 P、S、表面波或核心相位形成不同能量帶，就能比較其速度與路徑。
  • 看振幅與頻率：不同距離的振幅變化反映幾何擴散、衰減、散射與多路徑聚焦。

這張圖可以放在網頁最後當作總結：前面學的走時曲線、射線參數、相位命名與衰減，最後都會回到「如何讀一張真實 record section」。
